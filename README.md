# Government Scheme Eligibility Predictor

This project predicts whether a user is eligible for different Indian government schemes based on details like age, income, and category.

## Features
- Eligibility prediction for government schemes
- Simple rule-based logic
- Easy to extend with new schemes

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
